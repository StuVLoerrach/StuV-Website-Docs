# Banner Section

The Banner Section component displays prominent announcements or messages at the top of a page.

## Overview

Banner sections are typically used to highlight important information, promotions, or announcements. They are designed to be attention-grabbing and concise.

## Props

| Prop | Type | Description | Default |
|------|------|-------------|---------|
| `variant` | `string` | The variant of the banner to display | `"variant1"` |
| `title` | `string` | The main heading text | `""` |
| `description` | `string` | The descriptive text | `""` |
| `backgroundColor` | `string` | Background color of the banner | `"bg-primary"` |
| `textColor` | `string` | Text color of the banner | `"text-white"` |
| `showCloseButton` | `boolean` | Whether to show a close button | `true` |
| `position` | `"top" \| "bottom"` | Position of the banner | `"top"` |
| `fullWidth` | `boolean` | Whether the banner spans the full width | `true` |

## Variants

### Variant 1: Simple Banner

A basic banner with text and optional close button.

<BannerSectionBlock
  variant="variant1"
  title="New Feature Available!"
  description="Check out our latest update with exciting new capabilities."
  backgroundColor="bg-blue-600"
  textColor="text-white"
  showCloseButton={true}
/>

### Variant 2: Banner with Action

Banner with a call-to-action button.

<BannerSectionBlock
  variant="variant2"
  title="Limited Time Offer"
  description="Get 20% off all products until the end of the month."
  backgroundColor="bg-amber-500"
  textColor="text-black"
  buttonText="Shop Now"
  buttonUrl="/shop"
/>

### Variant 3: Banner with Icon

Banner with an icon for visual emphasis.

<BannerSectionBlock
  variant="variant3"
  title="Important Notice"
  description="Our system will be under maintenance on Saturday."
  icon="Bell"
  backgroundColor="bg-red-500"
  textColor="text-white"
/>

### Variant 4: Countdown Banner

Banner with a countdown timer.

<BannerSectionBlock
  variant="variant4"
  title="Sale Ends Soon"
  description="Don't miss out on our biggest sale of the year!"
  endDate="2025-12-31T23:59:59"
  backgroundColor="bg-purple-600"
  textColor="text-white"
/>

### Variant 5: Multi-step Banner

Banner with multiple steps or messages that can be navigated.

<BannerSectionBlock
  variant="variant5"
  steps={[
    { title: "Step 1", description: "Create your account" },
    { title: "Step 2", description: "Set up your profile" },
    { title: "Step 3", description: "Start exploring" }
  ]}
  backgroundColor="bg-green-600"
  textColor="text-white"
/>

### Variant 6: Cookie Consent Banner

Banner specifically designed for cookie consent notices.

<BannerSectionBlock
  variant="variant6"
  title="Cookie Notice"
  description="We use cookies to enhance your browsing experience."
  acceptButtonText="Accept All"
  rejectButtonText="Reject Non-Essential"
  settingsButtonText="Cookie Settings"
  position="bottom"
  backgroundColor="bg-gray-800"
  textColor="text-white"
/>

## Usage

```tsx
export default function HomePage() {
  return (
    <main>
      <BannerSectionBlock
        variant="variant1"
        title="Welcome to our site!"
        description="Explore our new features and content."
        backgroundColor="bg-blue-600"
        textColor="text-white"
      />
      
      {/* Rest of your page content */}
    </main>
  );
}
```

## Accessibility

- All banners are announced to screen readers using appropriate ARIA roles
- Close buttons have accessible labels
- Color contrast meets WCAG AA standards
- Keyboard navigation is fully supported
