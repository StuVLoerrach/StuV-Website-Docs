# Banner Section

The Banner Section component displays prominent announcements or messages at the top of a page.

## Overview

Banner sections are typically used to highlight important information, promotions, or announcements. They are designed to be attention-grabbing and concise. The appearance and functionality of the banner are determined by the `type` you select.

## Props

| Prop | Type | Description | Default |
|------|------|-------------|---------|
| `type` | `'variant1' \| 'variant2' \| ...` | Select the layout variant for this banner section. | `null` |
| `heading` | `string` | Main heading for the banner section. | `""` |
| `description` | `string` | Short description or value proposition for the banner. | `""` |
| `ctaLink` | `object` | Link for the call-to-action button. Appears in most banner variants. | `null` |
| `image` | `string \| Media` | Select an image to be displayed in the banner for **Variant 2**. | `null` |
| `variant` | `'default' \| 'destructive'` | Select the alert style for the banner (e.g., for error states). | `"default"` |
| `endDate` | `string` | Set the date when this banner will stop being displayed. This is only applicable for **Variant 5**. | `null` |

## Variants

### Variant 1: Simple Banner

A basic banner with a heading and description.

<BannerSectionBlock
  blockType="bannerSection"
  type="variant1"
  heading="New Feature Available!"
  description="Check out our latest update with exciting new capabilities."
  variant="default"
/>

### Variant 2: Banner with Action and Image

Banner with a call-to-action button and a background image.

<BannerSectionBlock
  blockType="bannerSection"
  type="variant2"
  heading="Limited Time Offer"
  description="Get 20% off all products until the end of the month."
  image="/images/sale-banner.jpg"
  ctaLink={{
    type: 'custom',
    customUrl: '/shop',
    label: 'Shop Now',
    appearance: 'outline'
  }}
/>

### Variant 3: Informational Banner

A standard layout for general announcements.

<BannerSectionBlock
  blockType="bannerSection"
  type="variant3"
  heading="Important Notice"
  description="Our system will be under maintenance on Saturday."
  variant="destructive"
/>

### Variant 4: Promotional Banner

Another layout variant for highlighting promotions or key messages.

<BannerSectionBlock
  blockType="bannerSection"
  type="variant4"
  heading="Our Biggest Sale"
  description="Don't miss out on our biggest sale of the year!"
/>

### Variant 5: Countdown Banner

This banner variant includes an `endDate` prop, intended for use with a countdown timer.

<BannerSectionBlock
  blockType="bannerSection"
  type="variant5"
  heading="Sale Ends Soon!"
  description="This offer expires at midnight."
  endDate="2025-12-31T23:59:59"
/>

### Variant 6: Generic Banner

A versatile banner layout for various use cases.

<BannerSectionBlock
  blockType="bannerSection"
  type="variant6"
  heading="We Value Your Feedback"
  description="Let us know how we can improve your experience."
  ctaLink={{
    type: 'internal',
    internalPath: '/feedback',
    label: 'Give Feedback'
  }}
/>

## Usage

```tsx
import { BannerSectionBlock } from './types'; // Assuming types are in a file
import { Page } from './types'; // Assuming a Page type exists

function HomePage() {
  const samplePageReference: Page = { /* ... page data */ };

  return (
    <main>
      <BannerSectionBlock
        blockType="bannerSection"
        type="variant1"
        heading="Welcome to our site!"
        description="Explore our new features and content."
      />
      
      {/* Example with a CTA linking to another page in the CMS */}
      <BannerSectionBlock
        blockType="bannerSection"
        type="variant2"
        heading="Learn More About Us"
        description="Discover our company's mission and values."
        ctaLink={{
          type: 'reference',
          reference: {
            relationTo: 'pages',
            value: samplePageReference
          },
          label: 'About Us'
        }}
      />
      
      {/* Rest of your page content */}
    </main>
  );
}
```

## Accessibility

- All banners are announced to screen readers using appropriate ARIA roles.
- Links within the `ctaLink` prop should have accessible labels.
- Ensure color contrast of rendered banners meets WCAG AA standards.
- Keyboard navigation is fully supported for interactive elements like the `ctaLink`.
