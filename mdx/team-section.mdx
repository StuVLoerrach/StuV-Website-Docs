# Team Section

The Team Section component showcases team members, staff, or key personnel in various layouts.

## Overview

Team sections help humanize your organization by showcasing the people behind your success. They build trust and connection with your audience by highlighting expertise and personalities.

## Props

| Prop | Type | Description | Default |
|------|------|-------------|---------|
| `variant` | `string` | The variant of the team section to display | `"variant1"` |
| `title` | `string` | The main heading text | `"Our Team"` |
| `subtitle` | `string` | The secondary heading text | `""` |
| `description` | `string` | The descriptive text | `""` |
| `teamMembers` | `TeamMember[]` | Array of team members to display | `[]` |
| `backgroundColor` | `string` | Background color of the section | `"bg-white"` |
| `textColor` | `string` | Text color of the section | `"text-gray-900"` |
| `layout` | `"grid" \| "carousel" \| "list"` | Layout style | `"grid"` |
| `columns` | `number` | Number of columns in grid layout | `3` |
| `showSocialLinks` | `boolean` | Whether to display social media links | `true` |
| `cardStyle` | `"minimal" \| "detailed" \| "hover"` | Style of team member cards | `"detailed"` |

  return (
    <main>
      <TeamSectionBlock
        variant="variant1"
        title="Our Amazing Team"
        subtitle="The People Behind Our Success"
        description="We're a diverse group of passionate individuals working together to create exceptional experiences."
        teamMembers={[
          {
            name: "John Smith",
            role: "Founder & CEO",
            image: "/images/team/john.jpg",
            bio: "John founded the company with a vision to revolutionize the industry through innovation and customer focus.",
            socialLinks: {
              linkedin: "https://linkedin.com/in/johnsmith",
              twitter: "https://twitter.com/johnsmith",
              email: "john@company.com"
            }
          },
          {
            name: "Jane Doe",
            role: "Head of Engineering",
            image: "/images/team/jane.jpg",
            bio: "Jane leads our engineering team with expertise in scalable systems and modern development practices.",
            socialLinks: {
              linkedin: "https://linkedin.com/in/janedoe",
              github: "https://github.com/janedoe"
            }
          },
          {
            name: "Mike Johnson",
            role: "Creative Director",
            image: "/images/team/mike.jpg",
            bio: "Mike brings creative vision to our products, ensuring beautiful and functional user experiences.",
            socialLinks: {
              linkedin: "https://linkedin.com/in/mikejohnson",
              behance: "https://behance.net/mikejohnson"
            }
          }
        ]}
        layout="grid"
        columns={3}
        showSocialLinks={true}
        cardStyle="hover"
        backgroundColor="bg-gray-50"
      />
    </main>
  );
}
\`\`\`

## Accessibility

- All images include descriptive alt text
- Social media links have proper labels
- Keyboard navigation is fully supported
- Focus indicators are clearly visible
- Screen readers can navigate team member information
- Color contrast meets WCAG AA standards
