# FAQ Section

The FAQ Section component displays frequently asked questions and their answers in various formats.

## Overview

FAQ sections help address common questions and concerns that users might have. They provide quick access to important information and can reduce support inquiries by addressing common issues proactively.

## Props

| Prop | Type | Description | Default |
|------|------|-------------|---------|
| `variant` | `string` | The variant of the FAQ section to display | `"variant1"` |
| `title` | `string` | The main heading text | `"Frequently Asked Questions"` |
| `subtitle` | `string` | The secondary heading text | `""` |
| `description` | `string` | The descriptive text | `""` |
| `faqs` | `FAQ[]` | Array of questions and answers | `[]` |
| `backgroundColor` | `string` | Background color of the section | `"bg-white"` |
| `textColor` | `string` | Text color of the section | `"text-gray-900"` |
| `layout` | `"accordion" \| "grid" \| "tabs"` | Layout style | `"accordion"` |
| `columns` | `number` | Number of columns in grid layout | `1` |
| `showSearchBar` | `boolean` | Whether to display a search bar | `false` |
| `categorized` | `boolean` | Whether FAQs are organized by category | `false` |
| `categories` | `Category[]` | Array of FAQ categories | `[]` |

## Usage

\`\`\`tsx
import { FAQSectionBlock } from "./faq-section";

export default function SupportPage() {
  return (
    <main>
      <FAQSectionBlock
        variant="variant3"
        title="Frequently Asked Questions"
        description="Find answers to common questions or search for specific topics."
        showSearchBar={true}
        faqs={[
          {
            question: "How do I get started?",
            answer: "Sign up for an account and follow our quick start guide to begin using the platform.",
            tags: ["getting started", "signup", "guide"]
          },
          {
            question: "Is my data secure?",
            answer: "Yes, we use industry-standard encryption and security measures to protect your data.",
            tags: ["security", "data", "privacy", "encryption"]
          },
          {
            question: "Can I integrate with other tools?",
            answer: "We offer integrations with popular tools like Slack, Google Workspace, and many others.",
            tags: ["integrations", "tools", "api", "connections"]
          }
        ]}
        backgroundColor="bg-white"
        layout="accordion"
      />
    </main>
  );
}
\`\`\`
