# Stats Section

The Stats Section component displays key metrics, achievements, or data points in visually appealing formats.

## Overview

Stats sections are perfect for showcasing important numbers, achievements, or key performance indicators. They help build credibility and communicate success metrics effectively.

## Props

| Prop | Type | Description | Default |
|------|------|-------------|---------|
| `variant` | `string` | The variant of the stats section to display | `"variant1"` |
| `title` | `string` | The main heading text | `""` |
| `subtitle` | `string` | The secondary heading text | `""` |
| `description` | `string` | The descriptive text | `""` |
| `stats` | `Stat[]` | Array of statistics to display | `[]` |
| `backgroundColor` | `string` | Background color of the section | `"bg-white"` |
| `textColor` | `string` | Text color of the section | `"text-gray-900"` |
| `layout` | `"horizontal" \| "grid" \| "vertical"` | Layout style | `"horizontal"` |
| `columns` | `number` | Number of columns in grid layout | `4` |
| `animated` | `boolean` | Whether to animate numbers on scroll | `true` |
| `showIcons` | `boolean` | Whether to display icons with stats | `false` |
| `iconStyle` | `"outline" \| "solid" \| "duotone"` | Style of icons | `"outline"` |

## Usage

\`\`\`tsx

export default function AboutPage() {
  return (
    <main>
      <StatsSectionBlock
        variant="variant2"
        title="Our Achievements"
        description="Numbers that showcase our commitment to excellence."
        stats={[
          {
            value: "5,000+",
            label: "Projects Delivered",
            icon: "CheckCircle",
            description: "Successfully completed projects"
          },
          {
            value: "98%",
            label: "Client Retention",
            icon: "Heart",
            description: "Clients who work with us again"
          },
          {
            value: "24/7",
            label: "Support Available",
            icon: "Clock",
            description: "Round-the-clock assistance"
          },
          {
            value: "10+",
            label: "Years in Business",

        ]}
        layout="grid"
        columns={4}
        showIcons={true}
        animated={true}
        backgroundColor="bg-gray-50"
      />
    </main>
  );
}
\`\`\`

## Accessibility

- Numbers are announced properly to screen readers
- Icons have appropriate aria-hidden attributes
- Trend indicators include accessible descriptions
- Color is not the only way to convey information
- Animation respects user's motion preferences
