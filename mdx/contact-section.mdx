# Contact Section

The Contact Section component provides various layouts for contact forms and information.

## Overview

Contact sections allow users to get in touch with your organization through forms, display contact information, and provide location details. These components are essential for creating effective communication channels on your website.

## Props

| Prop | Type | Description | Default |
|------|------|-------------|---------|
| `variant` | `string` | The variant of the contact section to display | `"variant1"` |
| `title` | `string` | The main heading text | `"Contact Us"` |
| `subtitle` | `string` | The secondary heading text | `""` |
| `description` | `string` | The descriptive text | `""` |
| `formConfig` | `FormConfig` | Configuration for the contact form | `defaultFormConfig` |
| `contactInfo` | `ContactInfo` | Contact information to display | `{}` |
| `mapLocation` | `MapLocation` | Location data for map display | `null` |
| `backgroundColor` | `string` | Background color of the section | `"bg-white"` |
| `textColor` | `string` | Text color of the section | `"text-gray-900"` |
| `layout` | `"standard" \| "split" \| "centered"` | Layout style | `"standard"` |
| `showSocialLinks` | `boolean` | Whether to display social media links | `false` |
| `socialLinks` | `SocialLink[]` | Array of social media links | `[]` |

## Usage

\`\`\`tsx
import { ContactSectionBlock } from "./contact-section";

export default function ContactPage() {
  return (
    <main>
      <ContactSectionBlock
        variant="variant2"
        title="Get in Touch"
        description="We'd love to hear from you. Fill out the form below and we'll get back to you as soon as possible."
        formConfig={{
          fields: [
            { name: "name", label: "Full Name", type: "text", required: true },
            { name: "email", label: "Email Address", type: "email", required: true },
            { name: "phone", label: "Phone Number", type: "tel", required: false },
            { name: "subject", label: "Subject", type: "text", required: true },
            { name: "message", label: "Your Message", type: "textarea", required: true }
          ],
          submitButtonText: "Send Message"
        }}
        contactInfo={{
          email: "contact@yourcompany.com",
          phone: "+1 (555) 123-4567",
          address: "123 Business Street, Suite 100, City, State 12345"
        }}
        showSocialLinks={true}
        socialLinks={[
          { platform: "twitter", url: "https://twitter.com/yourcompany" },
          { platform: "facebook", url: "https://facebook.com/yourcompany" },
          { platform: "instagram", url: "https://instagram.com/yourcompany" },
          { platform: "linkedin", url: "https://linkedin.com/company/yourcompany" }
        ]}
        layout="split"
      />
    </main>
  );
}
\`\`\`
